# smart-document-generator

AI-powered document editor for building professional CVs and resumes with live preview and export.

## Features
- AI chat assistant to build and edit your CV in real time
- Live CV preview with instant updates
- Customizable themes, fonts, and color schemes
- Export to PDF
- Built with React + Webpack + Babel

## Getting Started
```bash
npm install
npm start
```

App runs at `http://localhost:3001`
